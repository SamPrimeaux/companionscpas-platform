Help & Support

Adding Images

  1. Click "Add New Animal" or "Edit" on an existing animal
  2. Click on the image upload area
  3. Select your image file (JPG, PNG, or WebP)
  4. The image will upload automatically to our secure storage
  5. Add any context or notes in the description field for website refinement

Collaboration Features

Use the description and notes fields to add context about each animal's story, special needs, or presentation preferences. This information helps ensure the best presentation on the public website. Any context you provide will be reviewed and refined to create compelling animal profiles.

💡 Pro Tip: The more context you provide, the better we can showcase each animal's unique personality and story!
